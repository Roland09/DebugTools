# DebugTools
 
Tools for developing and debugging Unity
  
#### DebugBounds

Visualize the axis aligned and non axis aligned bounds of a gameobject.

Usage

* Drag DebugBounds.cs on a gameobject and select the bounds you want to see as Gizmo
